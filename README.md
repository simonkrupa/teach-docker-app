## Install

Clone the repo and install dependencies:

```bash
git clone --depth 1 --branch main https://github.com/simonkrupa/teach-docker-app.git your-project-name
cd your-project-name
npm install
npm run rebuil
npm start
```

## Packaging for Production

To package apps for the local platform:

```bash
npm run package
```
